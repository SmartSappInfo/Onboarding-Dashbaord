### 21.4 Add workspace filter to message history view ✅

**Implementation:**

- Uses `where('workspaceIds', 'array-contains', activeWorkspaceId)` query
- Shows only messages visible to the active workspace

**Code:**
```typescript
const logsQuery = useMemoFirebase(() => {
    if (!firestore || !activeWorkspaceId) return null;
    return query(
        collection(firestore, 'message_logs'), 
        where('workspaceIds', 'array-contains', activeWorkspaceId),
        orderBy('sentAt', 'desc'), 
        limit(200)
    );
}, [firestore, activeWorkspaceId]);
```
